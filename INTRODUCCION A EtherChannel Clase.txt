INTRODUCCIÓN A ETHER CHANNEL PRÁCTICA PKT DE INTRO

Todo lo trabajaríamos como grupos 

Comando para el grupo una vez accesado a la interfaz o interfaces
channel-group 1 mode active


---- LACP
	En ambos lados
	interface range GigabitEthernet 0/1-2
	channel-group 1 mode active
	exit
	interface port-channel 1
	switchport mode trunk
	switchport trunk native vlan 1
	switchport trunk allowed vlan 1


Verificar con 

show EtherChannel
show interfaces port-channel 1   (No soportado por Packet Tracer)
show EtherChannel summary
show EtherChannel port-channel
show interfaces EtherChannel
show running-config
show running-config | begin interface port-channel 
show interface trunk
